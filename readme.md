# Apprentissage de Pygame et de la Programmation Orientée Objet (POO)

Ce projet est conçu pour découvrir les bases de [Pygame](https://www.pygame.org/) et de la Programmation Orientée Objet (POO) en Python.

## 📖 Objectif
- Apprendre à utiliser Pygame pour créer des jeux simples.
- Comprendre les principes de la POO en les appliquant dans un contexte ludique.

## 🛠️ Fonctionnalités
- Gestion d'une fenêtre de jeu.
- Création et gestion d'entités (joueurs, objets, ennemis, etc.) via la POO.
- Gestion des événements (clavier, souris).
- Simulation de mouvements et interactions simples.

- Python 3.10+
